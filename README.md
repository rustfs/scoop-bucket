# RustFS Scoop Bucket

[![Tests](https://github.com/rustfs/scoop-bucket/actions/workflows/ci.yml/badge.svg)](https://github.com/rustfs/scoop-bucket/actions/workflows/ci.yml)
[![Excavator](https://github.com/rustfs/scoop-bucket/actions/workflows/excavator.yml/badge.svg)](https://github.com/rustfs/scoop-bucket/actions/workflows/excavator.yml)

Scoop bucket for RustFS command-line tools.

## Available Manifests

- `rc` - A Rust S3 CLI client for S3-compatible object storage

## Installation

```pwsh
scoop bucket add rustfs https://github.com/rustfs/scoop-bucket
scoop install rustfs/rc
```
